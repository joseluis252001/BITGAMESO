Politica de Privacidad

Ultima actualizacion: Mayo 2026 — Esta politica aplica a bitgameso.com

BITGAMESO es una plataforma educativa sin fines de lucro creada como proyecto escolar. No vendemos datos, no mostramos publicidad personalizada y no compartimos tu informacion con terceros.

1. Informacion que recopilamos

Cuando te registras en BITGAMESO recopilamos:

  • Nombre de usuario
  • Correo electronico
  • Progreso del juego (monedas, mascotas, inventario)

No recopilamos datos de pago, documentos de identidad ni informacion sensible.

2. Como usamos tu informacion

  • Para crear y gestionar tu cuenta
  • Para guardar tu progreso en el juego en la nube
  • Para enviarte correos de confirmacion y recuperacion de contrasena
  • Para recibir reportes de bugs que nos ayuden a mejorar el juego

3. Almacenamiento de datos

Tus datos se almacenan en Supabase, una plataforma segura con cifrado en reposo y en transito. El progreso del juego tambien se guarda localmente en tu navegador (localStorage).

4. Cookies y almacenamiento local

BITGAMESO usa localStorage del navegador para guardar tu sesion y progreso. No usamos cookies de rastreo ni publicidad.

5. Servicios de terceros

  • Supabase — base de datos y autenticacion
  • Vercel — hospedaje del sitio web
  • Resend — envio de correos transaccionales

Estos servicios tienen sus propias politicas de privacidad y cumplen con estandares de seguridad internacionales.

6. Tus derechos

Tienes derecho a:

  • Acceder a tu informacion personal
  • Solicitar la eliminacion de tu cuenta y datos
  • Corregir informacion incorrecta

Para ejercer cualquiera de estos derechos contactanos por correo.

7. Seguridad

Implementamos medidas de seguridad que incluyen:

  • HTTPS con certificado SSL en todo el sitio
  • Claves API protegidas en variables de entorno del servidor
  • Row Level Security (RLS) en la base de datos
  • Headers de seguridad HTTP (X-Frame-Options, HSTS, etc.)

8. Menores de edad

BITGAMESO es una plataforma educativa apta para todas las edades. No recopilamos conscientemente datos de menores de 13 anos sin consentimiento parental.

9. Cambios a esta politica

Podemos actualizar esta politica ocasionalmente. Te notificaremos por correo si hay cambios significativos.
